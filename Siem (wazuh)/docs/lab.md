# Laboratório SIEM - Wazuh

## Objetivo

Validar a coleta e análise de eventos Windows utilizando Wazuh.

---

## Escopo

Neste laboratório serão monitorados:

- Login bem sucedido
- Login inválido
- Criação de usuário
- Alteração de senha
- Exclusão de usuário
- Execução de PowerShell
- Execução de CMD
- Instalação de software

---

## Critérios de Sucesso

O laboratório será considerado funcional quando:

- O agente permanecer conectado.
- Os eventos chegarem ao Dashboard.
- Os eventos forem identificados corretamente.
- Os testes forem documentados.

---

## Ferramentas

- Ubuntu Server
- Windows 10
- Wazuh
- VirtualBox
